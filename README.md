# simulations
Making some simple simulations of different things using python.

## Folders:

### 'random_walk' 
Contains a python file with random walk in 2 dimensions and pictures of possible realizations it. 

### 'harmonic_oscillators'
Contains a python file that has a function which simulates and graphs the motion of a classical (non-quantum) Simple Harmonic Oscillator (SHO), whose behavior is guided by Hook's law: f = - kx (f stands for force, not frequency: the angular frequency of the oscillator is given in the file as w). There are also pictures with examples of the plotted oscillations as a function of time with varying parameters.\
In future, models of other oscillators, like SHO with damping and maybe a string pendulum will be added.\

### laser beams
In the folder 'pictures', 'bgb' stands for Bessel-Gaussian Beam, 'bb' is for Bessel Beam and 'gb' is for Gaussian Beam. The intensity of the beams is given by the square of the amplitude in arbitrary units (the distribution is what matters).
